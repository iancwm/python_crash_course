# Technology Stack

## 1. Programming Language

*   **Python**: The primary programming language for all development, scripting, and data analysis tasks within the crash course.

## 2. Development Environment

*   **Jupyter Notebooks**: The interactive computing environment used for creating, sharing, and executing documents containing live code, equations, visualizations, and narrative text. This facilitates an interactive and exploratory learning experience.

## 3. Package Management

*   **uv**: A highly performant and modern Python package installer and resolver, used for efficient dependency management and environment setup.

## 4. Key Libraries & Frameworks

*   **Pandas**: A fast, powerful, flexible, and easy-to-use open-source data analysis and manipulation tool, built on top of the Python programming language. Essential for data cleaning, transformation, and analysis.
*   **NumPy**: The fundamental package for scientific computing with Python. It provides powerful N-dimensional array objects and sophisticated (broadcasting) functions.
*   **Scikit-learn**: A comprehensive and widely-used machine learning library in Python, offering simple and efficient tools for data mining and data analysis. It covers various classification, regression, and clustering algorithms.
*   **Matplotlib/Seaborn**: Powerful libraries for creating static, interactive, and animated visualizations in Python. Matplotlib provides the foundation, while Seaborn builds on top of it to offer a higher-level interface for drawing attractive and informative statistical graphics.
