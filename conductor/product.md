# Product Guide: Python Crash Course

## 1. Core Concept

This project is a comprehensive, hands-on crash course in Python, with a strong focus on data science applications. It is structured as a series of Jupyter Notebooks designed to guide a learner ("Rick") from foundational Python concepts to advanced data analysis and machine learning projects.

## 2. Target Audience

The primary audience is an individual learner, "Rick," who is likely new to Python or data science and needs a structured, practical introduction to the key tools and methodologies in the field.

## 3. Project Goals

*   **Educate:** Provide a clear, step-by-step educational path covering core Python programming.
*   **Demonstrate:** Showcase practical data science workflows, including Exploratory Data Analysis (EDA), data cleaning, visualization, and classification.
*   **Empower:** Equip the learner with the foundational skills to use libraries like Pandas, Numpy, and likely Scikit-learn for their own data projects.

## 4. Key Components

*   **Session Notebooks:** Sequentially ordered lessons on atomic data types, data structures, flow control, functions, OOP, and regular expressions.
*   **Advanced Topics:** Dedicated notebooks for essential data science libraries: NumPy and Pandas.
*   **Applied Projects:** Three distinct projects that apply the learned concepts to real-world datasets:
    1.  **Pokemon EDA:** Exploratory data analysis on Pokemon stats.
    2.  **Cannabis Classification:** A machine learning classification task.
    3.  **Bank Churn Prediction:** A predictive modeling project.
